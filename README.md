# Career Recommendation System

## About the Project

This is a Career Recommendation System developed using Python, Flask, and Machine Learning. The main purpose of this project is to help students and job seekers find suitable career options based on their education, skills, interests, certifications, and communication skills.

The system predicts the **top 3 career recommendations** and also provides a **Skill Gap Analysis** for the best recommended career. This helps users understand which skills they should improve to become more prepared for their desired career.

## Features

* Career recommendations based on user input
* Top 3 predicted career options
* Confidence score for each recommendation
* Skill Gap Analysis
* Simple and user-friendly web interface

## How to Run

1. Clone this repository.
2. Install the required libraries:

```bash
pip install flask pandas numpy scikit-learn
```

3. Run the project:

```bash
python main.py
```

4. Open your browser and visit:

```text
http://127.0.0.1:5000
```
